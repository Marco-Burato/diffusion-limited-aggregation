# diffusion-limited-aggregation

A simple **Diffusion-Limited Aggregation (DLA)** simulation and analysis in a Jupyter notebook.

## Contents
- `Diffusion_Limited_Aggregation.ipynb`: simulation + basic analysis (e.g., morphology / growth patterns)

## How to run
1. Create a Python environment (recommended)
2. Install dependencies (typical): numpy, matplotlib, jupyter
3. Open the notebook and run all cells

## Output
The notebook generates visualizations of the aggregate and related observables.
